# Shell Scripts para SRE/DevOps

Este repositório contém uma coleção de scripts shell que facilitam diversas tarefas rotineiras de SREs e DevOps.
## Começando

Para utilizar os scripts deste repositório, clone-o em seu sistema local ou servidor onde você deseja executar os scripts.

```bash
git clone https://github.com/seuusuario/shell-scripts.git
cd shell-scripts
```

### Pré-requisitos

Certifique-se de que você tem `bash` instalado em seu sistema, pois é necessário para executar os scripts. Além disso, alguns scripts podem requerer ferramentas específicas, que serão listadas junto aos scripts.

### Instalação

Não há necessidade de instalação específica para a maioria dos scripts. No entanto, para scripts que requerem permissões de execução, use:

```bash
chmod +x nome-do-script.sh
```

## Uso

Abaixo estão exemplos de como executar alguns dos scripts disponíveis. Certifique-se de revisar cada script para entender as opções disponíveis e configurações necessárias.

```bash
# Executar um script para monitoramento de logs
./assume-role.sh

# Executar um script para backup de dados
./backup-dados.sh
```
